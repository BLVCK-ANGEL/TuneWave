## This code ate every Discord music bot for an evening snack 

### Setting Up

1. Clone the repo:
   ```bash
   git clone https://github.com/teamkronix/kronix-music-bot.git
   cd kronix-music-bot
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the bot:
   ```bash
   node index.js
   ```
